# Change: Add stable version downloads and align About page fields

## Why
Users need a clear place to download stable releases and a concise About page with standard app information.

## What Changes
- Add a release list that shows stable releases with version, publish date, platform, summary, and a download action.
- Update the About page to show only version number, author, website, copyright, update channel.

## Impact
- Affected specs: version-downloads, about-page
- Affected code: Core release fetch, GUI version list UI, About window content, tests
